# 물가 상승 생존 게임: 한 달 예산 지키기

중학교 3학년 사회② 경제 단원 「우리나라 경제와 세계화 - 물가 변동과 실업」 수업에서 사용할 수 있는 교사용 정적 웹앱입니다. 교사가 노트북에서 실행한 뒤 교실 스크린에 띄우고, 학생 모둠의 토의 결과를 화면에 입력하며 진행합니다.

UI는 BOOONG 디자인 시스템의 브랜드 블루, 세리프 본문, 산세리프 UI, 얇은 카드 보더, 콜아웃형 수업 패널 스타일을 따릅니다.

## 실행 방법

1. 프로젝트 폴더에서 `index.html`을 브라우저로 엽니다.
2. 별도 서버, 로그인, 데이터베이스, Firebase 설정은 필요하지 않습니다.
3. 새로고침하면 진행 상황은 초기화됩니다.

## 파일 구조

```text
/index.html
/css/style.css
/js/app.js
/js/data.js
/README.md
```

## 수업 활용 방법

- 교사는 화면을 빔프로젝터 또는 TV에 띄웁니다.
- 학생들은 모둠별로 역할 카드를 확인하고 매 라운드 선택지를 토의합니다.
- 라운드 화면의 토의 타이머를 사용해 45초, 60초, 1분 30초 토의 시간을 운영할 수 있습니다.
- 20분 수업은 기본값인 60초 토의를 권장합니다. 지연되면 25분 안쪽에서 마무리하는 흐름으로 운영합니다.
- 교사는 각 모둠 카드에서 학생들이 고른 선택지를 클릭합니다.
- 모든 모둠의 선택이 끝나면 결과를 보여 주고 “핵심 정리” 한 문장만 먼저 읽습니다.
- 결과 화면의 토론 질문을 활용해 역할별 유리함과 불리함을 비교하게 합니다.
- 최종 결과 화면에서 “결과 저장”을 누르면 같은 브라우저에 최근 수업 결과가 임시 저장됩니다.

## 게임 진행 순서

1. 시작 화면에서 수업을 시작합니다.
2. 수업 설명 화면에서 물가, 물가 지수, 인플레이션, 분배 효과를 확인합니다.
3. 모둠 수를 2~8모둠 사이에서 정하고 이름을 입력합니다.
4. 역할 카드가 자동 배정됩니다.
5. 총 5라운드의 물가 상승 상황을 진행합니다.
6. 각 라운드 결과 화면에서 지표 변화와 해설을 확인합니다.
7. 최종 결과 화면에서 점수, 예산, 안정도, 위험도를 비교합니다.

## 교사용 운영 팁

- 선택지를 고르기 전에 “이 역할은 물가 상승 때 무엇이 가장 부담일까?”를 먼저 묻습니다.
- 선택 대기 표시와 진행률 막대를 보며 아직 입력하지 않은 모둠을 확인합니다.
- “교사용 진행 노트 / 학습지 기록 포인트”를 펼쳐 라운드별 발문과 학습지 작성 위치를 확인합니다.
- 20분 수업에서는 라운드마다 발표를 1모둠만 받고, 자세한 해설은 필요할 때만 펼쳐 봅니다.
- 결과 화면에서는 점수가 높은 모둠만 보지 말고 안정도, 예산, 위험도를 함께 비교하게 합니다.
- 최종 결과의 라운드별 변화 추이를 보며 어떤 선택이 예산, 안정도, 위험도에 누적 영향을 주었는지 질문합니다.
- 금리 상승 라운드에서는 대출자와 예금자의 결과 차이를 강조하면 개념 이해가 쉽습니다.
- 임대료 상승 라운드에서는 건물 소유자와 자영업자, 세입자의 이해관계 차이를 토론하게 합니다.
- 최종 화면의 “캡처용 보기”를 누르면 상단 조작 버튼이 숨겨져 수업 정리나 활동 기록용으로 캡처하기 좋습니다.

## 교과 개념 연결

- 물가: 시장에서 거래되는 재화와 서비스의 평균적인 가격 수준
- 물가 지수: 물가 수준의 변화를 측정하는 지표
- 인플레이션: 전반적인 물가가 지속적으로 상승하는 현상
- 실질 구매력: 같은 돈으로 살 수 있는 상품과 서비스의 양
- 분배 효과: 물가 상승이 경제 주체의 소득, 자산, 부채 상태에 따라 서로 다른 영향을 주는 현상
- 예산 조정: 제한된 소득 안에서 소비, 저축, 투자, 상환의 우선순위를 정하는 선택

## 향후 확장 아이디어

- 교사가 라운드별 물가 상승률을 직접 조정하는 기능
- 저장된 여러 반의 결과를 비교하는 기능
- 토론 질문을 라운드마다 여러 개 중 무작위로 제시하는 기능
- 저장 결과를 CSV 또는 인쇄용 문서로 내보내는 기능
- 인쇄용 활동지 또는 정리표 자동 생성
